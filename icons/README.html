SVG icons for the KrishiMitra PWA. For production, convert these to PNG using a tool like:

npx svgexport icons/icon-512.svg icons/icon-512.png 512:512

Available Icons

512x512
192x192

Required Sizes

For full PWA support, generate these PNG sizes:

  • 72x72 - Android (low)
  • 96x96 - Android (medium)
  • 128x128 - Android (high)
  • 144x144 - Android (x-high)
  • 152x152 - iOS
  • 192x192 - Android (xx-high) / Chrome
  • 384x384 - Android (xxx-high)
  • 512x512 - Chrome / Splash screen

Quick Generate Script

Create a script to generate all sizes:

# Install sharp-cli globally
npm install -g sharp-cli

# Generate all sizes from 512 SVG
sharp -i icon-512.svg -o icon-72.png resize 72 72
sharp -i icon-512.svg -o icon-96.png resize 96 96
sharp -i icon-512.svg -o icon-128.png resize 128 128
sharp -i icon-512.svg -o icon-144.png resize 144 144
sharp -i icon-512.svg -o icon-152.png resize 152 152
sharp -i icon-512.svg -o icon-192.png resize 192 192
sharp -i icon-512.svg -o icon-384.png resize 384 384
sharp -i icon-512.svg -o icon-512.png resize 512 512
